## Create image and button onbutton click hide the image
```
<!DOCTYPE html>
<html lang="en">
<head>
<title>Show/Hide image with jQuery</title>
<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.3.1/jquery.min.js"></script>
<script type="text/javascript">
$(document).ready(function(){
  $('#btn1').on("click",function(e){
   $('#myImg').toggle('slow');
  });
});
</script>
</head>
<body>
  <img src="img.jfif" alt="Show/Hide Image" id="myImg"/>
  <button type="button" id="btn1">Show/Hide Image</button>
</body>
</html>
```

## Create a paragraph, write 5 line about you. on mouse over hide paragraph and on mouse out show paragraph

```
<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js"></script>
<script>
$(document).ready(function(){
  $("#p1").mouseenter(function(){
    $("#p1").hide();
     $("#p1").mouseleave(function(){
    $("#p1").show();
  });
  });
});
</script>
</head>
<body>

<p id="p1">Hello I am Tushar Dhage. I Studied my degree from YCCE Nagpur
           and i have job experience of 1.5 year in wipro lightning nagpur. 
           now i m doing diploma in pg-dac from cdac vita mumbai.</p>

</body>
</html>
```

##  Write your name in h1 tag with blue color on mouse enter it should display red border around it.

```
<!DOCTYPE html>
<html>
<head>

<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js"></script>
<script>
$(document).ready(function(){
  $("p").on({
    mouseenter: function(){
      $(this).css("background-color", "blue");
    }
  });
});
</script>

</head>
<body>

<style>
p.tad{
     color: red;
     }
</style>

<p class="tad">Tushar Dhage</p>

</body>
</html>
```

## Create a text box, on blur give grey color to textbox
```
<!DOCTYPE html>
<html>
<head>
<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js"></script>

<script>
$(document).ready(function(){
  
  $("input").blur(function(){
    $(this).css("background-color", "grey");
  });
});
</script>

</head>
<body>

<input type="text" name="fullname">
</body>
</html>
```

## Create a text box when you type in it the color of text should be green

```
<!DOCTYPE html>
<html>
<head>
<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js"></script>
<script>
$(document).ready(function(){
  $("input").focus(function(){
    $(this).css("color", "green");
  });
  
});
</script>
</head>

<input type="text" name="fullname"><br>
```
